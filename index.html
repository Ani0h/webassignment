Introduction to HTML by Annie Mzembe

  • Doctype

    • The Doctype is a tag that tells the browser what kind of document its reading.

    Example of the code

    <!doctype html>

  • html

    • HTML stands for hypertext markup language. The HTML tag acts is a root (top level code) for all code. All the code is written within its tag, as such all code is a descendant of this root element.

    Example of the code

    <html> </html>

  • head

    • The head tag provides information about the document, such as its language, title and style sheets. This information is read and used by the browser not the person viewing the web page.

    Example of the code

     <head></head>

  • body

    • The body holds all the information (content) that will be displayed on the webpage.

    Example of the code

     <body></body>

  • title

    • The title is the name of the webpage. This is read by the browser and can also be viewed by the peson reading the webpage. The title is usaully displayed in the tab of of a browser.

    Example of the code

    <title></title>

  • p

    • This is a paragraph tag. it is used to sort text into different paragraphs.The content is written between the openning and closing "p" tags.

    example of the code

    <p>This is a Paragraph</p>

  • tables and caption

    • The Caption tag acts as a heading for the table. The table tag allows the presentation of data and information into rows and columns. The design of the table is dependent on the value of borders, the cellspacing and cell padding.

    example of the code

    <table width="200" border="2" cellspacing="5" cellpadding="20"> 

    <caption> CAPTION </caption>

     <tbody> 

    <tr>

     <th scope="col">Table Heading</th> 

    <th scope="col">Table Heading</th>

     </tr> 

    <tr> 

    <td>Table data</td>

     <td>Table data</td>

     </tr> 

    </tbody>

     </table> 

    Working example

    CAPTION
    Table Heading Table Heading
    Table data Table data

  • lists

    • lists are used to sort information or data in a list form. there are different types of lists. The information is listed in list items tag (li) wrapped in either the "ul" or "ol" tag. Another form of lists are definition lists (dl), these go in hand with definition terms (dt).

    example of the code

    ordered list

    <ol>

     <li>Ordered List</li>

     <li>Unordered List</li> 

    <li>Definition List</li>

     </ol> 

    <ul> 

    <li>Ordered List</li> 

    <li>Unordered List</li> 

    <li>Definition List</li>

     </ul> 

    Working example

    1. Ordered List
    2. Unordered List
    3. Definition List

  • qoute and blockqoute

    • A quote is used to place inverted commas around the qoute. the quote tag is usually used inline.

    <blockquote> I was angry with my friend; I told my wrath, my wrath did end. I was angry with my foe: I told it not, my wrath did grow. </blockquote>

  • meta

    • The HTML element represents metadata that cannot be represented by other HTML meta-related elements, like base, link, script, style or title. Such as telling the Browser the language of the HTML document.

    example of the code

  • Special symbols and character entities

    The special symbol tags are used to represent symbols or currency. They are represented by using ampersand, a tag name or number followd by a semi-colon.

    • Example of the code
    Character Name  Example of the code 
     Copyright  <p>&copy;</p>
     Trade Mark  <p>&#8482;</p>
     Registered Trademark  <p>&reg;</p>
     Euro  <p>&#8364;</p>
     Non-breaking Space  <p>&nbsp;</p>
     Less than  <p>#8250;</p>
    Greater than   <p>#8249;</p>
     Ampersand  <p>&amp;</p>
